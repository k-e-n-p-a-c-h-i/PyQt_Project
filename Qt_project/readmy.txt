Справочник домашней ректификации.

Выполнил ученик 2 курса: Суедов Владимир Алексеевич

Идея проекта заключается в том, чтобы создать удобное приложения,
которое станет незаменимым для тех кто занимается домашней ректификацией.
Оно должно иметь возможность добавлять, изменять и удалять записи рецептов и ректификаций, изображений.
Также в приложении должны быть реализованы основные инструменты для расчетов, необходимых при процессе ректификации.
Должен быть реализован функционал управления приложением с помощью клавиш клавиатуры.
Переход к сайту с законодательной базой.

Для реализации заданного функционала, была спроектирована база данных: “rectification.db” включающая в себя 2 таблицы,
отвечающих за хранение данных о рецептах и ректификациях. Было спроектировано 14 классов виджетов и 1 класс ошибки.
Для инициализации каждого из 14 классов использовались ui файлы.
Как для добавления, так и для чтения, редактирования и удаления записей были созданы отдельные формы,
наследуемые от родительских форм. Для задания фона каждого окна была написана отдельная функция,
которая применялась при инициализации каждого класса.
Работа с файлами (изображения) была реализована с помощью функционала библиотеки “os”.
Для перехода к сайту использовался функционал библиотеки “webbrowser”

В данном проекте были реализованы такие технологии как: работа с файлами, с изображениями, обработка событий, наследование.
Были использованы такие библиотеки как: os, sqlite3, sys, webbrowser, Pillow, PyQt5.
